# 🧠 Second Brain — Home

> [!tip] Your single entry point
> Start here. Capture fast, organize later. Everything is reachable in one or two clicks.

## ⚡ Quick capture
- 📥 **[[Inbox/Inbox|Inbox]]** — dump anything here, sort it later
- 📅 **[[Daily/Daily|Today's note]]** — log, journal, scratchpad
- 🗒️ New atomic idea → use the **[[Templates/Permanent Note|Permanent Note]]** template

## 🗺️ Navigate by Map of Content
Maps of Content (MOCs) are hand-curated hubs that link related notes together.
- **[[Maps of Content/Maps of Content|All Maps of Content]]**

## 📂 The PARA system
Everything lives in one of four buckets, ordered by actionability:

| Folder | What goes here | Index |
|--------|----------------|-------|
| 🎯 **Projects** | Active efforts with a goal + deadline | [[Projects/Projects\|open]] |
| 🌱 **Areas** | Ongoing responsibilities, no end date | [[Areas/Areas\|open]] |
| 📚 **Resources** | Reference material & topics of interest | [[Resources/Resources\|open]] |
| 🗄️ **Archive** | Anything inactive / completed | [[Archive/Archive\|open]] |

Plus two thinking spaces:
- 🗒️ **[[Notes/Notes|Notes]]** — atomic, evergreen permanent notes (your Zettelkasten)
- 🤖 **[[Memory/Memory|Memory]]** — ClawAgent's auto-managed notes

## 🔁 The workflow
```text
Capture → Inbox / Daily
   ↓
Clarify → Is it actionable?
   ↓
Organize → Project / Area / Resource
   ↓
Distill → Write a Permanent Note (Notes/)
   ↓
Connect → Link it into a Map of Content
```

## 🔎 Find anything
- 🗂️ **[[Index]]** — master catalog: browse by type, tag, recency, or find orphans
- 🏷️ **[[Tags]]** — the tag taxonomy (controlled vocabulary)
- Press `Ctrl/Cmd + O` to jump to any note by name
- Press `Ctrl/Cmd + Shift + F` to search all content
- Browse by tag: `#moc`, `#project`, `#area`, `#resource`, `#permanent`, `#literature`

> [!note] Using Dataview?
> If you install the **Dataview** community plugin, the index notes will auto-populate
> with live lists. Without it, the manual links below each index still work fine.
