# 🧠 Second Brain (Obsidian Vault)

A personal knowledge base built for **quick capture** and **easy organization**.
Synced via Obsidian Git — accessed by ClawAgent.

👉 **Open [`Home.md`](Home.md) in Obsidian to start.**

## The system at a glance
This vault combines three proven methods:

- **PARA** for organizing by actionability — `Projects/`, `Areas/`, `Resources/`, `Archive/`
- **Zettelkasten** for thinking — atomic, evergreen notes in `Notes/`
- **Maps of Content (MOCs)** for navigation — hubs in `Maps of Content/`

Capture is frictionless via `Inbox/` and `Daily/`; templates live in `Templates/`.

## Folder structure
| Folder | Purpose |
|--------|---------|
| `Home.md` | Master dashboard & entry point |
| `Inbox/` | Quick capture — sort later |
| `Daily/` | Daily notes (journal, log, scratchpad) |
| `Projects/` | Active efforts with a goal + deadline |
| `Areas/` | Ongoing responsibilities (no end date) |
| `Resources/` | Reference material & topics of interest |
| `Notes/` | Atomic permanent notes (Zettelkasten) |
| `Maps of Content/` | Curated navigation hubs |
| `Archive/` | Inactive / completed items |
| `Memory/` | ClawAgent's auto-managed notes |
| `Templates/` | Note templates for every type |
| `Attachments/` | Images & media |

## Workflow
`Capture → Clarify → Organize → Distill → Connect`
See `Home.md` for the full loop.

## Setup in Obsidian
1. Install the **Obsidian Git** community plugin and point it at this repo.
2. Set an auto-commit interval (recommended: 5 minutes).
3. *(Recommended)* In Settings → Community plugins, turn off Restricted Mode, then
   **Browse → install Dataview** (and optionally **Calendar**). Dataview is already
   pre-enabled and configured in `.obsidian/` — once the files download it just works,
   and all the index tables (Index, Tags, Projects, etc.) render live.

Daily Notes and Templates core plugins are pre-configured in `.obsidian/`.
