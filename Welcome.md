# ClawAgent Vault

This vault is synced with ClawAgent via Obsidian Git.

## Setup
1. Install the **Obsidian Git** community plugin in Obsidian
2. Point it at: `https://github.com/jmiller18899-lab/obsidian-vault`
3. Set auto-commit interval (recommended: 5 minutes)

## Structure
- `Daily/` — daily notes
- `Projects/` — project notes
- `Memory/` — ClawAgent memory notes (auto-managed)
- `Inbox/` — quick capture

ClawAgent reads and writes notes here via the GitHub API.
