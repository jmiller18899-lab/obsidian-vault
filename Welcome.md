# 👋 Welcome to your Second Brain

This vault is synced with ClawAgent via Obsidian Git.

## ⭐ Start here
Open **[[Home]]** — it's the dashboard that links to everything.

## How it's organized
- **Capture:** [[Inbox/Inbox|Inbox]] and [[Daily/Daily|Daily notes]]
- **Organize (PARA):** [[Projects/Projects|Projects]] · [[Areas/Areas|Areas]] · [[Resources/Resources|Resources]] · [[Archive/Archive|Archive]]
- **Think:** [[Notes/Notes|Notes]] (atomic permanent notes)
- **Navigate:** [[Maps of Content/Maps of Content|Maps of Content]]
- **Templates:** see the `Templates/` folder

## Setup
1. Install the **Obsidian Git** community plugin in Obsidian.
2. Point it at: `https://github.com/jmiller18899-lab/obsidian-vault`
3. Set auto-commit interval (recommended: 5 minutes).
4. *(Optional)* Install **Dataview** + **Calendar** for live dashboards.

ClawAgent reads and writes notes here via the GitHub API, primarily under `Memory/`.
